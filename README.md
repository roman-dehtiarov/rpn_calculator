# rpn_calculator - is a command line reverse polish notation calculator
# How to run
1. Clone this repository
2. Install node if missing
3. ```npm install```
4. ```npm run start```

# Usage
You can write RPL expressions in one line

``` > 2 3 + ```

or in multiple lines

```
> 1
> 2
> +
```
# Examples

```
> 5 
5
> 8
8
> +
13
```

------

```
> 5 5 5 8 + + -
-13.0
> 13 +
0.0
```
